# What Problem Does Salesforce Solve?

Salesforce solves problems related to:
- Managing large customer data
- Tracking sales and leads
- Communication between teams
- Customer support management
- Manual business processes

It helps companies work more efficiently and improve customer relationships.

---

# What is CRM?

CRM stands for **Customer Relationship Management**.

CRM is a system used by businesses to:
- Manage customer information
- Track interactions
- Improve customer satisfaction
- Increase sales productivity

---

#  What is an Object in Salesforce?

An Object in Salesforce is a database table used to store information.

### Example:
- Account
- Contact
- Opportunity

Example:
An **Account** object stores company details.

---

# Salesforce Basic Objects

## Account
Represents a company or organization.

### Examples:
- Amazon
- Infosys
- Apollo Hospital

---

## Contact
Represents a person related to an account.

### Examples:
- Customer
- Student
- Employee

---

## Opportunity
Represents a potential business deal or sales process.

### Examples:
- Product purchase
- College admission
- Hospital treatment package

---

# Difference Between Salesforce Admin and Developer

| Salesforce Admin | Salesforce Developer |
|-----------------|---------------------|
| Manages users and settings | Builds custom applications |
| Creates reports and dashboards | Writes Apex and Lightning code |
| Configures Salesforce features | Develops automation and integrations |
| Uses clicks more than code | Uses programming skills |
