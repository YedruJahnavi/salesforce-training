# Platform Basics

## What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to build, customize, and manage business applications. It provides tools for storing data, automating workflows, creating apps, and managing customer relationships without needing complex infrastructure.

The Salesforce Platform helps developers and administrators:
- Build custom applications
- Manage business data
- Automate processes
- Create reports and dashboards
- Improve customer and employee experience

---

## Salesforce Components

## App

An App in Salesforce is a collection of tabs, objects, and features grouped together for a specific business purpose.

### Example:
A College Management App can contain:
- Student records
- Faculty details
- Course information
- Attendance management

---

## Object

An Object in Salesforce is like a database table used to store information.

### Types of Objects:
- Standard Objects
- Custom Objects

### Examples:
- Account
- Contact
- Student (Custom Object)

Each object contains records and fields.

---

## Tab

A Tab is used to access objects, records, and features in Salesforce easily through the user interface.

### Example:
- Student Tab
- Course Tab
- Attendance Tab

Tabs help users navigate different sections of the application.

---

# Difference Between Configuration and Coding

| Configuration | Coding |
|---------------|--------|
| Uses clicks instead of code | Uses programming languages |
| Easier for admins | Used by developers |
| Faster implementation | More customization possible |
| Uses tools like Flow and Process Builder | Uses Apex and Lightning Components |
| Suitable for simple automation | Suitable for complex business logic |
